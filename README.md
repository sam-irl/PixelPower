# PixelPower
A 2D game where 8-bit people fight in an 8-bit arena
----
## Quickstart
**Prerequisites:**
- Git
- Java (JRE + JDK >= 1.7.0)

`git clone https://github.com/sam-irl/PixelPower.git`

`cd PixelPower`

`javac *`

`java Pixel.java`

## Development
PixelPower uses gitflow. PRs welcome.
